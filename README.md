This repository contains the comprehensive **metadata** for the **ChIP-Hub v1** database.

**ChIP-Hub** is a large-scale, integrative platform designed specifically for the plant research community to explore the plant regulome. It provides a standardized resource for visualizing and analyzing ChIP-seq data across diverse plant species.
